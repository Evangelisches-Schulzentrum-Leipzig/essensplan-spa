# DuD Essensplan Display

A simple Single Page Application (SPA) for displaying menus from our food provider's API, designed for signage displays.  
It uses the API provided by [RCS (Richter Computer Systemhaus GmbH)](https://rcs.de) for their [OBS system](https://www.rcs.de/rcs-obs-online-bestellsystem/).

## Usage

Open `index.html` in your browser and provide the following parameters as URL search params:

- `requestUrl`
- `referrerUrl`
- `mandantId`
- `speiseplanNr`

Example:
```url
index.html?requestUrl=...&referrerUrl=...&mandantId=...&speiseplanNr=...
```
